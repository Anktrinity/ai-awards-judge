# Event Tech Awards: AI Edition, Updated Judging Report

## Rubric note
I rescored the current extracted-text batch using the same 0-100 framework across Innovation, Impact, Execution, and Results, for a total out of 400. I applied the calibration memo by rewarding genuine originality, staying disciplined on Results where proof was thin or overly portfolio-level, and using evidence traceability plus category fit as the main tie-breakers.

This update reflects the expanded intake of **19 extracted entries**. The first three categories are unchanged because their entrant set did not change. The major movement is in **Most Innovative AI Application in Events**, which expanded from 3 entries to 10.

- **No change in the first three categories.** The entrant pools there did not expand.
- **Most Innovative AI Application in Events changed substantially.**
  - **Rozie Synopsis remains #1.**
  - **Pointr moves from #2 to #5.**

- **Higher-volatility existing entries:** Voxo, MyGraduationClip, eBrain, InEvent, Rozie Synopsis.
- **Newer volatility watch entries in Most Innovative:** DNA Vision Pro, VOK DAMS, Piper, and EventWeb. Each has a strong concept, but some claims would be even easier to defend with fuller supporting dashboards, screenshots, or more named case material.

  - **VOK DAMS over Piper** on conceptual originality, despite Piper being slightly more mature operationally.

This updated report is more stable than the earlier provisional version because the expanded intake has now been incorporated.
