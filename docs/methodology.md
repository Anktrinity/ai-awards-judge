# Methodology

AI Awards Judge separates policy from execution.

## Layers

1. **Rubric**
   - scoring dimensions
   - score bands
   - tie-break order
   - fairness rules

2. **Prompting or evaluation logic**
   - how the model or reviewer is instructed
   - how evidence is summarized
   - how rationale is structured

3. **Outputs**
   - scorecards
   - category rankings
   - shortlist recommendations
   - master report

## Why this matters

This makes the system easier to audit, reuse, and defend.

- Change the rubric without rewriting the app
- Change the model prompt without changing policy
- Change the report format without changing scoring logic

## Intended use cases

- event technology awards
- AI innovation awards
- startup pitch competitions
- accelerator selection panels
- procurement shortlist reviews

## Guardrails

- Keep rankings provisional if intake is incomplete
- Separate cited evidence from inferred conclusions
- Preserve originals and extracted text for traceability
- Prefer defensible value over vague ambition
