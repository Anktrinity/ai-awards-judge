# Rubric

The default rubric uses four dimensions, each scored out of 100.

## Dimensions

### Innovation
How original and forward-thinking is the approach?

### Impact
What difference has this made, or what difference is it credibly expected to make?

### Execution
How well has this been implemented or developed?

### Results
What has been achieved, and what does the evidence show?

## Tie-breaks
When totals are close, prefer:
1. clearer evidence
2. stronger category fit
3. more distinctive use of AI
4. more defensible real-world value

## Adaptation
For another awards program, update:
- `config/rubric.yaml`
- `config/categories.yaml`
- `config/prompts.yaml`
